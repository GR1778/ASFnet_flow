the face of estimation errors.
   Finally, a Transformer-based fusion regressor integrates the refined multi-
modal features for 3D pose estimation. Notably, the proposed AMS and
UDE modules are highly modular and flexible, and can be readily integrated
into existing lifting-based frameworks, including both single-frame and multi-
frame variants. The efficacy of our depth-aware approach is demonstrated
in our experiments. Our framework not only sets a new state-of-the-art on
benchmark datasets, outperforming existing lifting-based and fusion meth-
ods, but also delivers a critical practical advantage: it exhibits remarkable
robustness, significantly mitigating the performance penalty typically associ-
ated with using efficient, low-precision depth estimators. These gains quanti-
tatively validate that directly tackling the dual problems of spatial misalign-
ment and depth noise is key to advancing depth-aware 3D pose estimation.
The main contributions of this work are summarized as follows:

   • A novel framework that synergistically combines adaptive sampling
     with uncertainty-aware enhancement to address the core challenges in
     depth-aware 3D pose estimation: spatial misalignment from imperfect
     2D poses and inherent noise in depth maps.

   • We introduce an Adaptive Multi-Sampling (AMS) module that progres-
     sively refines sampling locations through iterative correction to extract
     comprehensive pose-relevant features.

   • We introduce an Uncertainty-aware Depth Enhancement (UDE) mod-
     ule, which suppresses noise and enhances feature reliability by modeling
     depth feature distributions with quantified uncertainty.

   • Experiments on benchmark datasets for 3D human pose estimation,
     i.e., Human3.6M [12] and MPI-INF-3DHP [13], demonstrate that ours

2. Related Work

   Recent deep learning methods for monocular 3D human pose estimation
have advanced significantly. Early approaches followed two distinct paths:
some [14] utilized depth cameras to estimate 3D poses from depth images, but
their applicability was limited by hardware requirements; others [15] directly
regressed 3D poses from RGB images, yet were constrained by available 3D
motion capture data. With progress in 2D pose estimation [5, 16], the lifting
paradigm—first estimating 2D poses then converting to 3D—has emerged as
a prominent approach, simplifying the estimation pipeline.

2.1. Lifting-based Methods

   Lifting-based approaches simplify 3D pose estimation by leveraging off-
the-shelf 2D pose detectors trained on diverse datasets and lifting their out-
puts to 3D, a paradigm pioneered by [6]. To address depth ambiguity, meth-
ods model spatio-temporal dependencies using LSTMs [17] and CNNs [18].
Several works [8] further incorporate human skeletal constraints via graph
neural networks. Recently, transformer-based methods [7] have gained sig-
nificant attention, building on their success in vision tasks. The latest ad-
vancement employs diffusion models, which learn valid 3D pose distributions
to synthesize diverse samples. This powerful generative paradigm addresses
data scarcity and poor generalization by serving as probabilistic data aug-
mentors, enriching the training hypothesis space and enhancing robustness,
particularly in low-data regimes. For example, some works [19, 20, 21] con-
dition the reverse process on 2D keypoints for geometric consistency, while
others [22, 23] embed anatomical priors for biomechanical feasibility. The


                                      7
framework has also been extended for zero-shot cross-domain generalization
[24] and robust multi-frame video estimation [25].
   Despite advances, methods relying solely on 2D poses face the ineluctable
challenge of depth ambiguity. Our work directly addresses this by incorpo-
rating explicit robust depth features to guide the lifting model.

2.2. Fusion methods

   Beyond mainstream approaches, some methods integrate 2D poses with
image information to compensate for insufficient 2D cues. However, these
methods face several limitations: some bypass established 2D detectors [26],
increasing complexity while losing detector benefits; others employ subop-
timal fusion strategies, such as segmenting local patches [27] or using data-
sparse heatmaps [21]. While CA-PF [9] queries global features with 2D poses
and PSTR [28] improves training, these methods still struggle with the inher-
ent inverse problem and lack explicit depth cues from 2D-derived features.
Other approaches explicitly target depth estimation: some design specialized
modules for joint depth ranking [10], while [11] regresses depth from local
patches via LSTM. Yet these methods suffer from limited accuracy due to
simplistic fusion process and local processing. Alternatively, methods relying
on depth camera datasets [29] face cost and generalization barriers, restrict-
ing their applicability to general RGB imagery.
   In contrast to prior works, our method delivers reliable depth cues for
pose estimation without relying on depth cameras, achieved through an
uncertainty-aware depth enhancement mechanism.

3. Method

3.1. Overview
   Figure 2 illustrates our overall framework, with key notations summarized
in Table 1. Given RGB images I ∈ RB×H×W ×3 , we employ a pre-trained
2D pose detector to extract: 2D poses p ∈ RB×J×2 and multi-scale image
features {Hs ∈ RB×Hs ×Ws ×Cs }Ss=1 . A depth estimator extracts depth features
D ∈ RB×H×W ×Cfuse . The Multi-Modal Feature Alignment module then aligns
depth and image features with the 2D pose via Adaptive Multi-Sampling
(AMS), extracting pose-related features while suppressing background. To
mitigate depth noise, UDE enhances features by downweighting unreliable
regions using confidence scores. Finally, the Transformer Regressor fuses all
modalities via the Multimodal Interaction Transformer for final regression.
